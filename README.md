# HTML-based-authentication-system

This project contains a simple authentication flow built using only HTML.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- Proper redirection using anchor tags
- No CSS used
- No JavaScript used
- Pure HTML structure

## Flow

Login → Dashboard  
Register → Login  
Forgot Password → Reset Password → Login  
Dashboard → Logout → Login"# html-authentication-poc"
